デジタル ID の設定

デジタル ID は、PDF 文書の電子署名フィールドに署名したり、他のユーザにより自分だけが参照できるように保護された PDF 文書を開くのに必要になります。デジタル ID には、ユーザの電子署名の他に、自分自身を証明するために入力した情報も含まれています。まだデジタル ID を取得していない場合は、サードパーティの署名ハンドラから取得できます。または、Acrobat Self-Sign セキュリティハンドラを使用してデジタル ID を作成し、その証明書を他のユーザと共有することもできます。

Acrobat Self-Sign セキュリティハンドラを使用してデジタル ID を作成するには、文書／セキュリティ設定を選択します。左側の「デジタル ID」を選択し、右側の「ID を追加」をクリックして、デジタル ID を追加ダイアログボックスの指示に従います。
