확인 표시 및 주석 상태 추가

검토가 끝난 주석을 표시하는 등의 개인적인 용도로 주석에 확인 표시를 할 수 있습니다. 확인 표시를 해 두면 확인 표시 여부에 따라 주석을 표시하거나 숨길 수 있습니다. 주석의 상태를 다른 검토자에게 전달하려면 검토 상태 설정(없음, 승인됨, 거부됨, 취소됨 그리고 완료됨)을 사용합니다. 그러나 확인 표시는 다른 검토자에게는 표시되지 않습니다.

손 도구 를 선택합니다.

확인 표시를 선택하여 주석에 확인 표시를 추가합니다.
