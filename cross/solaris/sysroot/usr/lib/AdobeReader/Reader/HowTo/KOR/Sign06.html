서명 모양 변경

기본 설정 대화 상자를 열고 왼쪽에서 보안을 선택합니다.

모양 아래에 있는 새로 만들기를 클릭합니다.

서명 모양의 제목을 입력하고 다음 중 하나를 수행합니다.

디지털 서명 아이콘과 텍스트 구성 섹션에서 선택한 항목만 표시하려면 그래픽 없음을 선택합니다.

사용자가 지정한 그래픽을 표시하려면 가져온 그래픽을 선택합니다. 파일을 클릭하고 찾아보기를 클릭한 다음 파일을 선택하고 확인을 클릭합니다.

사용자의 이름을 디지털 ID 파일에 나타나는 대로 표시하려면 이름을 선택합니다.

서명에 표시할 텍스트 항목을 선택하고 확인을 클릭합니다.

다음 작업을 수행할 수도 있습니다.
