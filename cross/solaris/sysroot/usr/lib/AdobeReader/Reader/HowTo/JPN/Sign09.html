証明書情報の共有

電子署名を検証、または文書を暗号化して他のユーザへ送信するには、他のユーザがデジタル ID 証明書にアクセスできることが必要です。Acrobat Self-Sign セキュリティハンドラを使用してデジタル ID を作成した場合または他のユーザが証明書にアクセスする手段がない場合は、証明書を送付することができます。

文書／セキュリティ設定を選択します。

左側の「デジタル ID」を選択し、右側のデジタル ID ファイルを選択します。「証明書を書き出し」をクリックします。

証明書を電子メールで送信するか、または後で送信するためにファイルに保存するかを選択してから、「次へ」をクリックします。

手順に従って、ファイルを電子メールで送信または保存します。
