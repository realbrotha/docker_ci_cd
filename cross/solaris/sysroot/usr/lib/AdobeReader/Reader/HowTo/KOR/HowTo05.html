PDF 작성(온라인)

온라인으로 Adobe PDF 작성 기능은 온라인에서 원본 문서를 Adobe PDF 파일로 변환하는 데 사용할 수 있는 웹 기반 서비스입니다. 변환된 PDF 파일은 Adobe Reader만 있으면 누구나 볼 수 있습니다. Microsoft Office 파일, 그래픽 형식, 웹 페이지 등의 파일 형식을 Adobe PDF 파일로 변환할 수 있습니다. 종이 캡처 기능을 사용하여 스캔한 종이 문서를 검색 가능한 PDF 파일로 만들 수도 있습니다.

이 서비스를 사용하여 PDF 파일을 만들려면

Adobe ID와 암호를 사용하여 Create Adobe PDF Online 웹 사이트 http://createpdf.adobe.com(영문)에 로그인합니다. ID와 암호가 없으면 이 URL에서 등록할 수 있습니다.

Create Adobe PDF 페이지에서 Select A File을 클릭하고 파일을 엽니다.

원하는 옵션을 설정하고 Create PDF를 클릭합니다.
