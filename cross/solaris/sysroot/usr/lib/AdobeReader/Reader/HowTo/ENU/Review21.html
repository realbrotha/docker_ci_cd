Export comments to Word (Windows)

You can transfer comments and markups from a tagged PDF document to a Microsoft Word document. By integrating text edits from PDF files into the corresponding Word files, you can revise files in Word 2002 or later. For best results, you must first create the tagged PDF document in Word. This feature is not available if you use Acrobat in a browser window.

(Optional) Make sure that all comments are contained within a single PDF document. To do this, you can import comments by choosing Comment > Import Comments.

Choose Comments > Export Comments > To Word. If the original Word document cannot be found, select it from the dialog box, and then click Open.

Click Yes to add the comments.

Make sure that the appropriate files are selected, specify which comments you want to import, and then click Continue.
