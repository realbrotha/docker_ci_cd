Change signature appearance

Open the Preferences dialog box, and select Security on the left.

Under Appearance, click New.

Enter a title for the signature appearance, and then do one of the following:

Select No Graphic to display only the digital signature icon and items you select in the Configure Text section.

Select Imported Graphic to display a graphic that you specify. Click File, click Browse, select a file, and click OK.

Select Name to display your name as it appears in your Digital ID file.

Select any text items you want to appear in the signature, and then click OK.
