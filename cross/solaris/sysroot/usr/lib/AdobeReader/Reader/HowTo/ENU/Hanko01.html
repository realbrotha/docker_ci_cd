Create and import digital identity stamps

Acrobat provides several digital identity stamps that you can use to approve PDF documents. These stamps and others appear in the Stamps palette, which opens automatically in an approval workflow (or open the Stamps palette via the Stamps menu on the Commenting toolbar). If you prefer, you can import an image to create a custom digital identity stamp. You can use most image formats to do this, such as PDF, JPG, BMP, or PSD. Then, add your identity settings to the stamp to use it in approval workflows.

To create a custom stamp:

In the Commenting toolbar, choose Create Custom Stamp from the Stamps pop-up menu.

Click Browse to select the image file, click Select, and then click OK.

Enter a category name and a name for your stamp, and click OK.
