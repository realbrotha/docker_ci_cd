Print comments and markups

Print a summary by doing one of the following:

In Acrobat, choose File > Print With Comments Summary.

In a web browser, make sure the Comments List is open, and then choose Print Comments > Print Comments Summary.

Note: In Adobe Reader, you can print PDF documents with markups or stamps by choosing either Document With Stamps or Document With Markups from the Comments And Forms pop-up menu in the Print dialog box.

In the Summarize Options dialog box, select the layout, paper size, and other options you want, and then click OK.

In the Print dialog box, select the appropriate options for your printer, and then click OK or Print.
