Add drawing markups

The Drawing Markups toolbar includes tools for drawing shapes, adding text boxes, and creating pencil sketches.

To display the Drawing Markups toolbar, choose View > Toolbars > Drawing Markups.

On the Drawing Markups toolbar, do any of the following:

Select a drawing tool from the toolbar or from one of the pop-up menus on the toolbar, such as the Rectangle tool (from the Arrow tool menu), and then draw an object.

Select the Callout Box tool , click to create a callout box, and then double-click the box to type your comment.

Select the Dimensioning tool , and then drag to draw a line.

To learn more about creating any of these markups, search for the tool in Help.
