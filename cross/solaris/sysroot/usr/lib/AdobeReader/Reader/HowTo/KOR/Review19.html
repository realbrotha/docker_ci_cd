주석 표시 및 숨기기

주석의 유형, 작성자 또는 상태별로 주석을 표시하거나 숨길 수 있습니다. 그러나 숨겨진 주석은 주석 보내기를 선택할 때 다른 주석과 함께 전송되지 않습니다.

주석 달기 도구 모음이나 주석 목록의 표시 메뉴에서 표시할 주석 유형을 선택합니다. 예를 들어, 노트 주석만 표시되도록 하려면 표시 > 유형별로 표시 > 노트를 선택합니다.

모든 주석을 숨기려면 표시 > 모든 주석 숨기기를 선택합니다. 다시 표시하려면 표시 > 모든 주석 표시를 선택합니다.

특정 범주에서 숨긴 주석을 다시 표시하려면 해당 범주에 대한 모든... 명령을 선택합니다. 예를 들어, 특정 검토자의 주석만 표시한 경우 표시 > 검토자별로 표시 > 모든 검토자를 선택합니다.
