Save a copy of a review PDF file

When you open the PDF attachment for an email-based review, use File > Save to save a tracked copy of the PDF document to your hard drive. Adding comments to a tracked copy ensures that comments will be received by the review initiator and merged into the original PDF document when you send them. Only one copy of the PDF attachment is tracked.

Note: If you save the PDF attachment in Acrobat using File > Save A Copy, an untracked copy of the PDF document is created in the specified location; the email attachment remains the only tracked copy.
