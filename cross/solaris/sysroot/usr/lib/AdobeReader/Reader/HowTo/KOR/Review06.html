브라우저 기반 검토에 참가

Adobe Reader에서 검토 개시자가 보낸 FDF 파일을 엽니다. 파일을 열면 브라우저가 시작되고 해당 PDF 파일이 열립니다. 주석 달기 도구 모음의 도구를 사용하여 주석을 추가합니다.

참고: Adobe Reader에서는 PDF 문서를 인증해야만 주석 달기 도구가 나타납니다. 인증 과정에는 약간의 시간이 걸릴 수 있습니다. 그러나 기다리는 동안에도 PDF 문서를 볼 수는 있습니다.

주석 달기 도구 모음의 주석 보내고 받기를 클릭하여 온라인 문서에 주석을 업로드하고 다른 검토자의 주석을 확인합니다. 필요한 만큼 이 작업을 수행합니다.

다른 검토자의 주석을 읽으려면 손 도구 를 선택하고 주석을 두 번 클릭합니다.

주석 추가가 끝나면 주석 보내고 받기를 다시 클릭합니다.

참고: 검토하기 기본 설정에서 설정이 잘못되었거나 방화벽을 차단 해제하지 않은 경우 또는 네트워크 연결이 끊어진 경우에는 브라우저 기반 검토 과정에서 PDF 문서를 열 때 문제가 발생할 수 있습니다. 이 경우 네트워크 관리자에게 문의하여 설정을 적절히 수정하거나 문서를 오프라인으로 검토하십시오.
