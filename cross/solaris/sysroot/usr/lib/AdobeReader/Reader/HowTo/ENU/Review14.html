Group markups

If you create several comments or markups, you can group them together so they behave as a single markup. Any modifications you make to one markup, such as moving it or changing its properties, apply to all markups in the group. When you select multiple markups using the Ctrl (Windows) or Command (Mac OS) key, the Group option is available from the context menu.

Note: You cannot group text edits.
