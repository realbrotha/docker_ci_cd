Search PDF documents

You can use either the Find toolbar or the Search PDF window to locate any instance of a word, series of words, or part of a word in an Adobe PDF document. Pressing Ctrl+F (Windows) or Command+F (Mac OS) displays the Find toolbar, which lets you search in the current PDF document and navigate through the search results. Pressing Shift+Ctrl+F (Windows) or Shift+Command+F (Mac OS) displays the Search PDF window, which lets you search with additional criteria in the current PDF document or multiple PDF documents in the location you specify.
