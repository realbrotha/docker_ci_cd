인증서 정보 공유

디지털 서명을 확인하거나 다른 사람이 문서를 암호화할 수 있도록 하려면 사용자의 디지털 ID 인증서에 다른 사람이 액세스할 수 있어야 합니다. 자체 서명된 디지털 ID를 만들었거나 다른 사람이 사용자의 인증서에 액세스할 수 없는 경우에는 이를 전송할 수 있습니다.

문서 > 보안 설정을 선택합니다.

왼쪽에서 디지털 ID를 선택하고 오른쪽에서 디지털 ID 파일을 선택한 다음 인증서 내보내기를 클릭합니다.

서명 정보를 전자 메일로 보낼지 또는 나중에 전자 메일로 보낼 수 있도록 파일로 저장할지 여부를 선택하고 다음을 클릭합니다.

화면에 표시되는 단계에 따라 파일을 전자 메일로 보내거나 저장합니다.
