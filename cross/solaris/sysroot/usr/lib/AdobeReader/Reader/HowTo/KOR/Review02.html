텍스트 편집 모드 표시

텍스트 편집 주석은 삽입하거나 삭제하거나 바꿔야 하는 텍스트를 나타냅니다. 주석 달기 도구 모음에는 텍스트 편집 모드 표시 도구와 기타 텍스트 편집 옵션이 들어 있는 팝업 메뉴가 포함되어 있습니다.

주석 달기 도구 모음에서 텍스트 편집 > 텍스트 편집 표시 도구를 선택하고 확인을 클릭한 다음, 다음 중 원하는 작업을 수행합니다.

텍스트를 추가할 단어 사이를 클릭한 다음 텍스트를 입력합니다. 삽입 캐럿이 표시되고 해당 텍스트가 팝업 창에 추가됩니다.

텍스트를 선택하고 백스페이스 또는 Delete 키를 누릅니다. 텍스트에 줄이 그어져 삭제되어야 한다는 것을 나타냅니다.

텍스트를 선택하고 바꿀 텍스트를 입력합니다. 선택한 텍스트에 줄이 그어지고 삽입된 텍스트가 팝업 창에 표시됩니다.

텍스트를 선택하고 텍스트 편집 메뉴에서 선택한 텍스트 강조 또는 선택한 텍스트에 밑줄 긋기를 선택합니다.

다음 작업을 수행할 수도 있습니다.
